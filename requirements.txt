# Xpanel VPS Management System - Requirements

socket.io@^4.7.2             # Real-time communication
jsonwebtoken@^9.0.2          # JWT authentication
bcryptjs@^2.4.3              # Password hashing
cors@^2.8.5                  # Cross-origin resource sharing
dotenv@^16.3.1               # Environment variables
node-ssh@^13.1.0             # SSH client for server connections
ws@^8.14.2                   # WebSocket support
multer@^1.4.5-lts.1          # File upload handling
archiver@^6.0.1              # Archive creation
unzipper@^0.10.14            # Archive extraction

### Client Dependencies:
react@^18.2.0                # React framework
react-dom@^18.2.0            # React DOM rendering
react-router-dom@^6.3.0      # Client-side routing
socket.io-client@^4.7.2      # Socket.io client
axios@^1.4.0                 # HTTP client
styled-components@^6.0.7     # CSS-in-JS styling
framer-motion@^10.16.4       # Animations
react-icons@^4.10.1          # Icon library
recharts@^2.7.2              # Charts and graphs
@monaco-editor/react@^4.5.1  # Code editor
react-hot-toast@^2.4.1       # Toast notifications

### Development Dependencies:
nodemon@^3.0.1               # Development server auto-restart

## System Requirements

### Основной сервер (где запускается Xpanel):
- Node.js 16.0.0 или выше
- npm 8.0.0 или выше  
- RAM: 2GB минимум, 4GB рекомендуется
- Диск: 1GB свободного места
- ОС: Windows 10+, macOS 10.15+, Linux (Ubuntu 18+, CentOS 7+)
- Порты: 3001 (по умолчанию, настраивается)

### VPS серверы (управляемые через Xpanel):
- Linux: Ubuntu 18+, Debian 9+, CentOS 7+, RHEL 7+
- SSH доступ с правами sudo или root
- Python 3.6+ (устанавливается автоматически если отсутствует)
- RAM: 128MB свободной памяти минимум
- Диск: 50MB свободного места
- Интернет соединение для загрузки компонентов
- Открытые порты для SSH (22 по умолчанию) и агента (8888 по умолчанию)

## Автоматически устанавливаемые компоненты на VPS:

### Python пакеты (устанавливаются через pip):
psutil                       # Системный мониторинг
websockets                   # WebSocket сервер для агента
asyncio                      # Асинхронное программирование

### Системные пакеты (устанавливаются через apt/yum):
python3                      # Python интерпретатор
python3-pip                  # Python package manager
curl                         # HTTP client
wget                         # File downloader
git (опционально)            # Version control (для Git модуля)

## Браузерная совместимость:
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

## Сетевые требования:
- Исходящие соединения на порт 22 (SSH)
- Исходящие соединения на настраиваемый порт агента (8888 по умолчанию)
- Доступ к репозиториям пакетов Linux (для установки зависимостей)

## Рекомендации по безопасности:
- Использование HTTPS в продакшене
- Настройка firewall на VPS серверах
- Регулярное обновление системных пакетов
- Использование SSH ключей вместо паролей (планируется в будущих версиях)

## Примечания:
- Все зависимости устанавливаются автоматически через npm
- VPS агент устанавливается автоматически при добавлении сервера
- Поддерживается автоматическое обновление компонентов
- Совместимость с различными дистрибутивами Linux
